05. Frontend rdzen i komunikacja

Opis plikow odpowiedzialnych za uruchomienie frontendu i komunikacje z API.

Powrot do strony glownej

frontend/src/main.tsx

Plik inicjuje aplikacje oraz QueryClientProvider.

export const queryClient = new QueryClient({
  defaultOptions: { queries: { staleTime: 1000 * 60 * 5, retry: 1 } }
})

frontend/src/App.tsx

Plik definiuje routing i warstwe chronionych tras przez ProtectedRoute.

frontend/src/contexts/AuthContext.tsx

Context utrzymuje stan sesji i profil zalogowanego uzytkownika.

frontend/src/lib/api.ts

Wspolny klient HTTP (axios) z interceptor 401 i mechanizmem odswiezenia tokenu.

if (status === 401 && !originalRequest._retry) {
  originalRequest._retry = true
  await api.post('/auth/refresh')
  return api(originalRequest)
}
